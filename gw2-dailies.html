Daily Achievements App for Guild Wars 2

While working through the courses on Free Code Camp, I was working on multiple projects focusing on AJAX requests with public APIs. I'm an avid gamer and Guild Wars 2 player, I also know they have public APIs that the community can use to create web applications so I decided to create one for myself.

This app uses 3 different API end points to gather all of the data to display the daily achievement tasks and their descriptions for both today and tomorrow.

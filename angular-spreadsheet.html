Metrics Reporting Spreadsheet App

This is a fairly basic MEAN stack web app that I made as a prototype for another project I'm working on at my current role at Amazon.

This is my first project using the MEAN stack so it's pretty rough and incomplete, but also important as it helped solidify my understanding of the basics of the MEAN stack - like Express routing and REST APIs, Angular directives, and MongoDB data structuring and collections.

The database would be setup with a collection for each region, and an Object of data for each site. The cells are setup in such a way that when data is enterred, it saves only the data in the column enterred. The idea is that only one person would be enterring data at each site, so when you have multiple users enterring data on the app at the same time, there are no conflicts thanks to the non-blocking nature of NodeJS (Except, of course, unless you have 2 or more people enterring data for the same site at the same time).
